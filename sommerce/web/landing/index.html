Do I need to pay something extra?
No, for the platform you need to pay only for the monthly subscription. Extra payments can be only for provider or integrated services if you will choose a plan that is not for free.

Where can I buy a domain?
We allow this feature to our customers. You can buy your own domain during the shop order. We don’t charge any extra taxes, you pay only for a domain.

Do I need to know HTML?
We already took care about it and you don’t need to know anything about programming. You can build your website for some hours easily.

I'm not a company. Can I order a store?
Yes, sure. We provide our solution not only for companies. You can order a store being an individual.

Should I proceed with all orders by myself?
No, if you connected an SMM provider when adding a package to a product.

How will I know that I need to pay for the next month?
Don’t worry. We will generate an invoice and notify you before you need to pay so you won't miss anything.
